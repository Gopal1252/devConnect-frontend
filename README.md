# DevConnect Frontend

DevConnect is a Tinder-style networking platform for developers. Instead of dating, developers can:

- Create profiles showcasing their skills
- Browse other developers (like swiping on Tinder)
- Send connection requests to developers they're interested in
- Accept or reject incoming requests
- View their network of connections

## Frontend tect-stack

- React 18 - UI library
- Redux Toolkit - State management
- React Router v6 - Client-side routing
- Axios - HTTP client
- Tailwind CSS + DaisyUI - Styling
- Vite - Build tool
